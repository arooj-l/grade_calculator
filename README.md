# Grade Calculator (CLI)

A simple Python command-line application that allows users to enter marks dynamically and calculates:

- Total marks
- Average score
- Final letter grade

This project demonstrates:
- Loops
- Lists
- Error handling with try/except
- Conditional grading logic
- Basic input validation

---

## Features

- Users can enter unlimited marks
- Type `done` to finish input
- Handles invalid inputs safely
- Prevents division by zero if no marks are entered
- Automatically calculates grade based on average

---

## Grade Scale

| Average Score | Grade |
|--------------|-------|
| 90+          | A     |
| 80–89        | B     |
| 70–79        | C     |
| 60–69        | D     |
| Below 60     | F     |
